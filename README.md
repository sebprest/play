# Play

![Screenshot](https://github.com/sebprest/play/raw/master/screenshot.png)

Minimal CLI music player for macOS

## Requirements
macOS 1.7+ (tested with macOS 10.14)

## Usage
```
Usage: play [options] [file]
Options:
  --help | -h:
    Print this message.
```
